### Attractor Strength Scale
- **1.0**: Overwhelming pull (meanings cannot escape)
- **0.7**: Strong pull (most meanings drawn in)
- **0.5**: Moderate pull (influences nearby region)
- **Negative**: Repeller (meanings actively avoid)

## Mathematical Laws

Eight fundamental laws govern semantic space—like Newton's Laws for physics, but for meaning.

### 1. **Harmony Law**
```
H = 1 / (1 + d(coords, anchor))

where:
  H = harmony index [0, 1]
  d = Euclidean distance in 4D
  anchor = (1, 1, 1, 1)
```

**Interpretation**: Harmony is inversely proportional to distance from perfection.

**Properties**:
- H = 1 when coords = anchor (perfect harmony)
- H → 0 as distance → ∞
- H = 0.5 when distance = 1.0

**Significance**: Quantifies moral/aesthetic value. Every concept has an objective harmony score.

**Applications**:
- Moral evaluation: Is action A or B more harmonious?
- Aesthetic prediction: Will people find X beautiful? (Test: H > 0.60)
- Personal development: Track harmony trajectory over time

### 2. **Antonym Reflection Law**
```
opposite(C) ≈ 2×NE - C

where:
  C = concept coordinates
  NE = Natural Equilibrium (0.618, 0.414, 0.718, 0.693)
```

**Accuracy**: ~0.26 average error (90%+ accuracy)

**Interpretation**: Opposites reflect through Natural Equilibrium like light through a mirror.

**Examples**:

  - Predicted: (0.29, 0.23, 0.94, 0.69) — error: 0.01

**Significance**: Duality is geometric. "Opposite" has precise mathematical meaning.
